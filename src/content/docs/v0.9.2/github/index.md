The public nxus.SYSTEMS GitHub organization is
[github.com/nxus-SYSTEMS](https://github.com/nxus-SYSTEMS).

## Public Repositories

| Repository | What to use it for |
| --- | --- |
| [nxusKit SDK](https://github.com/nxus-SYSTEMS/nxusKit) | Source for the multi-language SDK covering LLM providers, rule engines, constraint solving, Bayesian networks, decision tables, the C ABI, and CLI/Bash workflows. |
| [nxusKit Examples](https://github.com/nxus-SYSTEMS/nxusKit-examples) | Runnable Rust, Go, Python, and CLI/Bash examples for common nxusKit integration patterns. |
| [nxus Docs](https://github.com/nxus-SYSTEMS/nxus-docs) | Source for this public documentation site, hosted at [docs.nxus.systems](https://docs.nxus.systems/). |
